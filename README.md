# Estudos — Sistemas IA-First e Arquitetura de Agentes

Este repositório organiza uma trilha progressiva de estudos sobre sistemas IA-first, arquitetura de agentes, governança, contexto, dados, segurança e operação de sistemas orientados por agentes.

O objetivo não é apenas estudar ferramentas de IA, mas compreender como projetar sistemas confiáveis que coordenam dados, contexto, agentes, workflows, governança, validação e pessoas.

## Organização inicial

A estrutura será dividida em duas coleções principais:

- `colecao-01-sistemas-ia-first/`: base acadêmica em três volumes.
- `colecao-02-formacao-arquiteto-agentes/`: formação profissional progressiva em oito fases.

Cada parte possui arquivos Markdown com objetivo, assuntos, metodologia, exercícios, entregáveis, critérios de avanço e prompt-base para abertura de um chat de estudo dedicado.
