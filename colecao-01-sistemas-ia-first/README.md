# Coleção 01 — Sistemas IA-First

## Propósito

Esta coleção organiza a base conceitual para compreender sistemas IA-first e arquitetura de agentes. O objetivo é construir uma progressão acadêmica que conecte história da computação, engenharia de software, dados, arquitetura, contexto, segurança, agentes e operação.

## Organização

- Volume 01 — Fundamentos dos Sistemas IA-First
- Volume 02 — Arquitetura de Sistemas Orientados por IA
- Volume 03 — Sistemas Operacionais de Agentes

## Como estudar

Estude os volumes em ordem. O primeiro volume estabelece a base histórica e técnica; o segundo aprofunda arquitetura, domínio, contexto, conhecimento e segurança; o terceiro integra esses fundamentos em sistemas de agentes e operação.

## Resultado esperado

Ao final da coleção, o estudante deve compreender como sistemas IA-first são concebidos, modelados, protegidos, coordenados e levados para produção.
