# Coleção 01 — Sistemas IA-First

Esta coleção organiza uma base acadêmica em três volumes para compreender sistemas IA-first, desde fundamentos até sistemas operacionais de agentes.

## Volumes

- Volume I — Fundamentos dos Sistemas IA-First
- Volume II — Arquitetura de Sistemas Orientados por IA
- Volume III — Sistemas Operacionais de Agentes

## Objetivo

Formar uma base conceitual sólida sobre dados, arquitetura, contexto, segurança, agentes, governança e produção.
