# Parte 14 — Governança

## Objetivo

Estudar governança aplicada a agentes para manter alinhamento, controle e auditabilidade.

## Assuntos principais

- Regras
- Políticas
- Guards
- Compliance
- Auditoria
- Controle operacional

## Como usar este diretório

- `prompt.md`: prompt-base para estudar governança de agentes.
- `exercicios.md`: espaço para exercícios de definição de regras e políticas.
- `notas.md`: espaço para registrar controles, padrões e observações.

## Entregáveis esperados

- Conjunto inicial de regras e políticas para agentes.
- Mapa de mecanismos de controle e auditoria.
- Síntese sobre alinhamento entre agentes e objetivos do sistema.

## Critérios de avanço

- Explicar a função de regras, políticas e guards.
- Definir critérios de auditoria e controle operacional.
- Relacionar governança com segurança e autonomia.

## Observações

Governança deve ser objetiva e operacional. Regras precisam orientar comportamento verificável.
