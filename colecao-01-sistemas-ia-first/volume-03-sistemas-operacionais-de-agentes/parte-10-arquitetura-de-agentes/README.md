# Parte 10 — Arquitetura de Agentes

## Objetivo

Entender os componentes internos de um agente e como eles se combinam para executar trabalho orientado por objetivos.

## Assuntos principais

- Planejamento
- Execução
- Memória
- Ferramentas
- Observação
- Feedback

## Como usar este diretório

- `prompt.md`: prompt-base para estudar arquitetura de agentes.
- `exercicios.md`: espaço para exercícios de decomposição e desenho de agentes.
- `notas.md`: espaço para registrar componentes, exemplos e dúvidas.

## Entregáveis esperados

- Descrição dos componentes internos de um agente.
- Mapa de fluxo entre planejamento, execução e feedback.
- Comparação entre agente, assistente e chatbot.

## Critérios de avanço

- Explicar o papel de cada componente de um agente.
- Identificar limites de autonomia e necessidade de supervisão.
- Relacionar arquitetura interna com objetivos operacionais.

## Observações

Agentes devem ser estudados como sistemas de execução orientados por contexto, ferramentas e controle.
