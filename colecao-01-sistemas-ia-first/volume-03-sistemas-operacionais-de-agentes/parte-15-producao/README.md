# Parte 15 — Produção

## Objetivo

Estudar como levar sistemas de agentes para produção com qualidade, rastreabilidade e observabilidade.

## Assuntos principais

- CI/CD
- Testes
- Observabilidade
- Logs
- Métricas
- Telemetria

## Como usar este diretório

- `prompt.md`: prompt-base para estudar operação em produção.
- `exercicios.md`: espaço para exercícios de validação, monitoramento e implantação.
- `notas.md`: espaço para registrar práticas, riscos e aprendizados.

## Entregáveis esperados

- Checklist de produção para sistemas de agentes.
- Mapa de sinais de observabilidade.
- Síntese sobre ciclo de operação e melhoria contínua.

## Critérios de avanço

- Explicar como testes e CI/CD reduzem riscos.
- Definir logs, métricas e telemetria relevantes.
- Relacionar produção com governança, segurança e qualidade.

## Observações

Produção exige disciplina operacional. Sistemas de agentes precisam ser monitoráveis e corrigíveis.
