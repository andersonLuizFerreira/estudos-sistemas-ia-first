# Parte 16 — Estudos de Caso

## Objetivo

Consolidar os conhecimentos dos volumes anteriores por meio da análise de soluções reais ou plausíveis.

## Assuntos principais

- ERP orientado por agentes
- SaaS multi-tenant
- Sistema especialista industrial
- Ecossistema empresarial
- Sistema operacional de agentes

## Como usar este diretório

- `prompt.md`: prompt-base para conduzir os estudos de caso.
- `exercicios.md`: espaço para exercícios de análise e desenho de soluções.
- `notas.md`: espaço para registrar conclusões, decisões e dúvidas.

## Entregáveis esperados

- Análise estruturada de cada caso.
- Propostas de arquitetura, dados, contexto, agentes, governança e segurança.
- Síntese final integrando os conceitos da coleção.

## Critérios de avanço

- Aplicar os conceitos anteriores em casos concretos.
- Justificar escolhas arquiteturais e operacionais.
- Identificar riscos, limitações e critérios de produção.

## Observações

Use os casos para integrar conhecimento. A qualidade está na coerência das decisões, não no volume de texto.
