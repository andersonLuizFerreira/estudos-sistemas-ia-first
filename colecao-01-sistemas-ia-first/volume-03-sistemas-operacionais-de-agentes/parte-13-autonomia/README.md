# Parte 13 — Autonomia

## Objetivo

Compreender autonomia supervisionada e os limites necessários para permitir ação sem perder controle.

## Assuntos principais

- Delegação
- Autoridade
- Responsabilidade
- Escalonamento
- Supervisão

## Como usar este diretório

- `prompt.md`: prompt-base para estudar autonomia em agentes.
- `exercicios.md`: espaço para exercícios de definição de limites e responsabilidades.
- `notas.md`: espaço para registrar riscos, decisões e dúvidas.

## Entregáveis esperados

- Matriz de autonomia, autoridade e responsabilidade.
- Exemplos de ações permitidas, bloqueadas e escalonadas.
- Síntese sobre supervisão humana e controle operacional.

## Critérios de avanço

- Diferenciar delegação de autorização irrestrita.
- Definir limites claros de ação para agentes.
- Explicar quando escalar uma decisão para supervisão.

## Observações

Autonomia deve ser tratada como capacidade controlada, não como ausência de governança.
