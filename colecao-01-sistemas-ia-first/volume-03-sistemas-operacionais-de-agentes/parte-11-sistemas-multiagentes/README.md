# Parte 11 — Sistemas Multiagentes

## Objetivo

Estudar como múltiplos agentes podem ser coordenados para dividir responsabilidades e executar trabalho complexo.

## Assuntos principais

- Planner
- Executor
- Validator
- Supervisor
- Router
- Coordenação
- Escalonamento

## Como usar este diretório

- `prompt.md`: prompt-base para estudar coordenação multiagente.
- `exercicios.md`: espaço para exercícios de desenho de papéis e fluxos.
- `notas.md`: espaço para registrar padrões, riscos e exemplos.

## Entregáveis esperados

- Mapa de papéis em um sistema multiagente.
- Fluxo de coordenação entre planejamento, execução e validação.
- Registro de riscos de comunicação e escalonamento.

## Critérios de avanço

- Diferenciar responsabilidades dos agentes principais.
- Explicar como ocorre coordenação entre agentes.
- Propor critérios para escalonar ou validar trabalho.

## Observações

Evite multiplicar agentes sem necessidade. A divisão de papéis deve resolver problemas reais de coordenação.
