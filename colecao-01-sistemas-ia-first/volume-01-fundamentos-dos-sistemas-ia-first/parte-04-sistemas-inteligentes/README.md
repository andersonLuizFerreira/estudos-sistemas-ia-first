# Parte 04 — Sistemas Inteligentes

## Objetivo

Entender a evolução dos sistemas inteligentes até os agentes modernos e diferenciar conceitos frequentemente confundidos.

## Assuntos principais

- Sistemas especialistas
- Machine learning
- Deep learning
- LLMs
- Assistentes
- Agentes

## Como usar este diretório

- `prompt.md`: prompt-base para estudar a evolução dos sistemas inteligentes.
- `exercicios.md`: espaço para exercícios comparativos e reflexões.
- `notas.md`: espaço para registrar definições, limites e exemplos.

## Entregáveis esperados

- Quadro comparativo entre IA, automação, chatbot, assistente e agente.
- Registro de limitações e contextos de uso de cada tecnologia.
- Síntese sobre a passagem de modelos inteligentes para agentes operacionais.

## Critérios de avanço

- Explicar como cada tecnologia funciona em alto nível.
- Identificar quando utilizar e quando não utilizar cada abordagem.
- Distinguir agente de chatbot ou assistente simples.

## Observações

O foco é clareza conceitual. Use exemplos, mas não transforme o README em aula técnica detalhada.
