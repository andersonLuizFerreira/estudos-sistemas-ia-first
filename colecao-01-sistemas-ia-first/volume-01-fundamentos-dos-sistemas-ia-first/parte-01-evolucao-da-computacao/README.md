# Parte 01 — Evolução da Computação

## Objetivo

Compreender a evolução histórica da computação até o surgimento de sistemas orientados por agentes.

## Assuntos principais

- História da computação
- Sistemas operacionais
- Redes e internet
- Sistemas cliente-servidor e web
- Cloud computing
- Automação
- Inteligência artificial
- Sistemas orientados por agentes

## Como usar este diretório

- `prompt.md`: prompt-base para abrir o chat de estudo desta parte.
- `exercicios.md`: espaço para registrar exercícios propostos e resolvidos.
- `notas.md`: espaço para anotações, dúvidas e sínteses pessoais.

## Entregáveis esperados

- Linha de raciocínio sobre a evolução dos sistemas computacionais.
- Mapa dos problemas resolvidos em cada estágio histórico.
- Reflexões sobre a transição de automação para agentes.

## Critérios de avanço

- Explicar a motivação histórica dos principais estágios da computação.
- Relacionar cada estágio a limitações do anterior.
- Conectar a evolução histórica com sistemas modernos orientados por IA.

## Observações

Priorize compreensão causal em vez de memorização de datas. A parte deve criar base para interpretar sistemas atuais.
