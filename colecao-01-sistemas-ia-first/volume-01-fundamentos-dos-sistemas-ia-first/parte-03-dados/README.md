# Parte 03 — Dados

## Objetivo

Estudar dados como base estrutural dos sistemas modernos e dos sistemas orientados por IA.

## Assuntos principais

- Modelagem relacional
- SQL
- Integridade
- Analytics
- BI
- Data Warehouse
- ETL
- Dados transacionais
- Dados analíticos

## Como usar este diretório

- `prompt.md`: prompt-base para orientar o estudo sobre dados.
- `exercicios.md`: espaço para exercícios de modelagem, consulta e análise.
- `notas.md`: espaço para observações, exemplos e dúvidas.

## Entregáveis esperados

- Comparação entre dados transacionais e analíticos.
- Exemplos de uso de dados em ERP, CRM, e-commerce e indústria.
- Síntese sobre a relação entre qualidade dos dados e qualidade da IA.

## Critérios de avanço

- Explicar integridade, modelagem e consulta de dados.
- Diferenciar uso operacional e analítico dos dados.
- Justificar por que dados sustentam sistemas inteligentes.

## Observações

Mantenha foco no papel arquitetural dos dados. Não transforme esta parte em estudo isolado de ferramentas.
