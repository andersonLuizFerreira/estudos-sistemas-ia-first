# Parte 07 — Engenharia de Contexto

## Objetivo

Estudar contexto como ativo principal de sistemas IA-first e como elemento arquitetural de coordenação.

## Assuntos principais

- Contexto
- Memória
- Conhecimento
- Recuperação de informação
- Contexto local
- Contexto global
- Contexto operacional
- Governança de contexto

## Como usar este diretório

- `prompt.md`: prompt-base para conduzir o estudo de engenharia de contexto.
- `exercicios.md`: espaço para exercícios de análise e desenho de contexto.
- `notas.md`: espaço para registrar estruturas, exemplos e dúvidas.

## Entregáveis esperados

- Classificação de tipos de contexto em sistemas reais.
- Mapa de fontes de conhecimento e memória.
- Síntese sobre governança de contexto.

## Critérios de avanço

- Explicar a diferença entre prompt, contexto, memória e conhecimento.
- Relacionar contexto com qualidade da decisão de sistemas IA-first.
- Propor formas de organizar contexto operacional.

## Observações

O objetivo não é escrever prompts melhores, mas compreender como sistemas sustentam contexto com consistência.
