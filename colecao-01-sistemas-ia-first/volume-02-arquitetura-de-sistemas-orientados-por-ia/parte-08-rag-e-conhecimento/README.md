# Parte 08 — RAG e Conhecimento

## Objetivo

Aprender como sistemas recuperam, organizam e utilizam conhecimento para apoiar aplicações de IA.

## Assuntos principais

- Embeddings
- Vetores
- Similaridade
- Chunking
- Retrieval
- Reranking
- Busca híbrida
- Bancos vetoriais

## Como usar este diretório

- `prompt.md`: prompt-base para estudar RAG e recuperação de conhecimento.
- `exercicios.md`: espaço para exercícios conceituais e práticos.
- `notas.md`: espaço para registrar conceitos, limitações e exemplos.

## Entregáveis esperados

- Explicação do fluxo de recuperação de conhecimento.
- Comparação entre busca vetorial, busca textual e busca híbrida.
- Registro de decisões sobre chunking, reranking e avaliação.

## Critérios de avanço

- Explicar o papel de embeddings e similaridade.
- Descrever um fluxo RAG com clareza.
- Identificar limitações e riscos de sistemas de recuperação.

## Observações

Trate RAG como arquitetura de conhecimento, não como recurso isolado de aplicação.
