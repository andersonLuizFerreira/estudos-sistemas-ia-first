# Fase 07 — Produção

## Objetivo

Aprender a colocar sistemas orientados por agentes em produção com disciplina técnica e operacional.

## Assuntos principais

- CI/CD
- Build
- Testes
- Lint
- Deploy
- Observabilidade
- Logs
- Métricas
- Telemetria

## Como usar este diretório

- `prompt.md`: prompt-base para estudar produção.
- `exercicios.md`: espaço para exercícios de pipeline, validação e monitoramento.
- `notas.md`: espaço para registrar práticas, falhas e melhorias.

## Entregáveis esperados

- Checklist de produção.
- Desenho de pipeline mínimo de CI/CD.
- Mapa de observabilidade com logs, métricas e telemetria.

## Critérios de avanço

- Explicar o papel de build, testes, lint e deploy.
- Definir sinais operacionais para monitoramento.
- Relacionar produção com governança e segurança.

## Observações

Produção não é apenas deploy. É capacidade contínua de validar, observar e corrigir o sistema.
