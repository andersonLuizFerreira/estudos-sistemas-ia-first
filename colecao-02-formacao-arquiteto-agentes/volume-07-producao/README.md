# Volume 07 — Produção

## Propósito

Este volume orienta a passagem de sistemas orientados por agentes para produção, com atenção a build, testes, deploy e observabilidade.

## Partes ou fases

- Fase 07 — Produção

## Pré-requisitos

O estudante deve compreender fundamentos, governança, coordenação multiagente, autonomia e segurança operacional.

## Resultado esperado

Ao final do volume, o estudante deve compreender práticas essenciais para operar sistemas orientados por agentes em ambientes reais.

## Critério de conclusão

O volume está concluído com qualidade quando o estudante consegue definir um fluxo mínimo de produção com validação, implantação, logs, métricas e telemetria.
