# Fase 05 — Autonomia Controlada

## Objetivo

Compreender autonomia supervisionada e definir limites operacionais para agentes.

## Assuntos principais

- Delegação
- Autoridade
- Responsabilidade
- Escalonamento
- Limites operacionais
- Supervisão humana

## Como usar este diretório

- `prompt.md`: prompt-base para estudar autonomia controlada.
- `exercicios.md`: espaço para exercícios de definição de limites e cenários.
- `notas.md`: espaço para registrar riscos, decisões e dúvidas.

## Entregáveis esperados

- Matriz de decisões autônomas, supervisionadas e bloqueadas.
- Critérios de escalonamento para supervisão humana.
- Síntese sobre responsabilidade em sistemas com agentes.

## Critérios de avanço

- Definir limites operacionais claros.
- Relacionar autoridade com responsabilidade.
- Explicar quando uma ação deve ser escalonada.

## Observações

Autonomia controlada exige fronteiras explícitas. O objetivo é permitir ação responsável.
