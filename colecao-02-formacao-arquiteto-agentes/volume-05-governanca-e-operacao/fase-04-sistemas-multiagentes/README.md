# Fase 04 — Sistemas Multiagentes

## Objetivo

Aprender coordenação de múltiplos agentes e divisão de responsabilidades em sistemas orientados por trabalho.

## Assuntos principais

- Planner
- Executor
- Validator
- Supervisor
- Router
- Especialização de agentes
- Delegação

## Como usar este diretório

- `prompt.md`: prompt-base para estudar sistemas multiagentes.
- `exercicios.md`: espaço para exercícios de desenho de papéis e fluxos.
- `notas.md`: espaço para registrar padrões, riscos e decisões.

## Entregáveis esperados

- Modelo de papéis de agentes.
- Fluxo de delegação, execução, validação e supervisão.
- Registro de critérios para especialização de agentes.

## Critérios de avanço

- Explicar a função de cada papel de agente.
- Definir como tarefas são delegadas e validadas.
- Identificar riscos de coordenação e supervisão.

## Observações

Coordenação deve reduzir complexidade operacional. Mais agentes não significam melhor arquitetura.
