# Fase 03 — Governança

## Objetivo

Estudar governança aplicada a projetos modernos, com foco em alinhamento, padrões e controle operacional.

## Assuntos principais

- Governança
- Regras
- Políticas
- Padrões
- Compliance
- Auditoria
- Controle operacional

## Como usar este diretório

- `prompt.md`: prompt-base para estudar governança.
- `exercicios.md`: espaço para exercícios de definição de regras e controles.
- `notas.md`: espaço para registrar padrões, dúvidas e exemplos.

## Entregáveis esperados

- Conjunto inicial de políticas e padrões.
- Mapa de controles operacionais.
- Síntese sobre compliance e auditoria em projetos modernos.

## Critérios de avanço

- Explicar o papel da governança na operação.
- Definir regras objetivas e verificáveis.
- Relacionar governança com qualidade, rastreabilidade e alinhamento.

## Observações

Governança deve orientar decisões práticas. Evite criar regras sem finalidade operacional clara.
