# Fase 01 — Fundamentos

## Objetivo

Construir os fundamentos necessários para compreender sistemas, processos, algoritmos, arquitetura básica e controle de versão.

## Assuntos principais

- Pensamento computacional
- Sistemas
- Processos
- Algoritmos
- Estruturas de dados
- Arquitetura básica de software
- Git
- GitHub
- Controle de versão

## Como usar este diretório

- `prompt.md`: prompt-base para conduzir o estudo dos fundamentos.
- `exercicios.md`: espaço para exercícios e práticas orientadas.
- `notas.md`: espaço para registrar dúvidas, exemplos e sínteses.

## Entregáveis esperados

- Síntese dos fundamentos estudados.
- Exercícios de pensamento computacional e organização de sistemas.
- Registro de práticas básicas de Git e controle de versão.

## Critérios de avanço

- Explicar conceitos fundamentais com clareza.
- Relacionar processos e sistemas a trabalho executável.
- Utilizar controle de versão como prática de organização.

## Observações

Esta fase deve consolidar base, não acelerar para ferramentas avançadas.
