# Fase 08 — Arquitetura Avançada

## Objetivo

Integrar todo o conhecimento adquirido para projetar sistemas complexos, plataformas IA-first e ecossistemas orientados por agentes.

## Assuntos principais

- Arquitetura corporativa
- Sistemas distribuídos
- Domain Driven Design
- Event-driven architecture
- Multi-tenant
- RAG
- Engenharia de contexto
- Runtime de agentes
- Sistemas operacionais de agentes
- Ecossistemas empresariais
- SaaS
- Plataformas IA-first

## Como usar este diretório

- `prompt.md`: prompt-base para conduzir o estudo de arquitetura avançada.
- `exercicios.md`: espaço para exercícios arquiteturais e estudos de caso.
- `notas.md`: espaço para registrar decisões, trade-offs e referências de estudo.

## Entregáveis esperados

- Proposta de arquitetura para uma plataforma orientada por agentes.
- Registro de trade-offs técnicos, operacionais e corporativos.
- Síntese final da formação como arquiteto de agentes.

## Critérios de avanço

- Integrar domínio, contexto, agentes, segurança e produção em uma arquitetura.
- Justificar decisões com base em restrições reais.
- Identificar riscos, dependências e critérios de evolução.

## Observações

Esta fase deve consolidar visão arquitetural. O estudante deve demonstrar capacidade de síntese e decisão.
